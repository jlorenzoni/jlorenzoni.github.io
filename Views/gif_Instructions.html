Git instructions

Local

  • Locally means that thingd are happening just on my computer.
  • Locally commits back up to a local repository and others can't access until we push to a remote

Remote

  • Remote menas your browser connects to an online server.
  • Remote depositories can be accessed by fellow team member (private repo) or publicly(public repo).

Local Setup

  1. Open your command line terminal
  2. Use the cd command to navigate to your folder, For example cd desktop/playground
  3. Type in git init. This will set up a repository(repo) for you locally.
  4. A message like this should pop up "Initialized empty gif repository in dir/filenale/stuff/here"

Local commits

  1. Make sure you're in the proper directory in the command line
  2. Enter git add . Note: This will add all the files that you have changed.
  3. Enter git commit -m "A note about the things you've changed". Note: The words in quotes can be anything
  4. Your code is now backed up locally
